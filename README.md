# Lead Management System

A role-based Lead Management System built with **Core PHP and MySQL**, 
enabling Admins and Agents to manage cases and leads efficiently.

## Technologies Used

- PHP (Core)
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Features

### Admin Panel
- Upload bulk cases via Excel file
- Manage cases (add, edit, delete)
- Follow-up on existing cases
- Assign cases to Agents
- Calendar to track case assignment dates

### Agent Panel
- View assigned cases
- Update case status and follow-ups
- Calendar view for assigned tasks

### Security
- Role-based access control (Admin & Agent)
- Secure authentication system
